# ✈️ Flight Booking Application

📌 Overview

This is a full-stack Flight Booking Application that allows users to search flights, book tickets, and manage bookings. It also includes an admin panel to manage flights and bookings.

---

🚀 Tech Stack

* Frontend: React (Vite)
* Backend: Node.js, Express.js
* Database: MySQL
* Authentication: JWT

---

## ✨ Features

👤 User Features

* User Registration & Login
* Search Flights
* Book Tickets
* View Booking History

🛠️ Admin Features

* Add / Update / Delete Flights
* Manage Bookings
* View Customer Details
